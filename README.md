# Лабораторная работа 9-10: Продвинутое ООП на Kotlin

## Описание
Лабараторная работа посвящена изучению продвинутых концепций объектно-ориентированного программирования в Kotlin.

## Структура проекта
Проект содержит примеры реализации продвинутых механизмов ООП, а также пояснение и ключевым концепуиям.

## Как запустить проект
1. Клонируйте репозиторий:
```bash
git clone <URL_репозитория>
```
2. Откройте проект в Intellig IDEA.
3. Запустите любой пример через контекстное меню или напрямую из `main`.

## Изученое в лабораторной работе:

## Геттеры и сеттеры
были использованы геттеры и сеттеры для управления доступом к свойствам классов.
с помощью сеттеров выполнялась проверка значений и ограничение допустимого
диапазона данных.

## Инкапсуляция
Инкапсуляция применялась для защиты данных от некорректного изменения.
Для этого использовались модификаторы доступа и приватные сеттеры.

## Data-классы
Data-классы использовались для хранения данных, таких как ресурсы.
Они позволили автоматически получить методы `toString()`, `equals()` и `copy()`.

## Абстрактные классы
Абстрактные классы применялись для создания общей логики модулей аванпоста.
Они задают структуру и обязательные методы для классов-наследников.

## Интерфейсы
Интерфейсы использовались для описания общего поведения различных классов.
Это позволило реализовать полиморфизм и работать с объектами через общий интерфейс.

## Пример использования интерфейса
Модули аванпоста реализуют интерфейс действий и взаимодействуют с менеджером
ресурсов, не зная его внутренней реализации.
